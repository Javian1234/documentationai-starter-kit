## Getting Started

To start using Fixit Jamaica Documentation:

1. Sign in or create an account at `https://docs.fixitjamaica.com`
2. Create a new documentation project from your dashboard
3. Choose a project name and description
4. Click "Create Project"

## Next Steps

- Add your first documentation page
- Invite collaborators via the project settings
- Use the sidebar to organize your docs

For more help, contact support anytime.
