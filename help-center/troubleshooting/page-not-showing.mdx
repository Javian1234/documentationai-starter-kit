---
title: Documentation Page Not Showing
description: What to do if a documentation page does not appear.
---

## Problem

- Added a new page but it does not appear in the sidebar
- Page link returns 404 error

## Quick Fix

<Steps>
  <Step title="Refresh the dashboard">
    Click your browser's refresh button or press `Ctrl+R`.
  </Step>
  <Step title="Check publishing status">
    Open project settings and verify the page is published, not in draft.
  </Step>
  <Step title="Clear browser cache">
    Try loading the site in a private/incognito window.
  </Step>
</Steps>
