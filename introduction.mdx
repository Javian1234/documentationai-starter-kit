## Overview

Fixit Jamaica Documentation helps you create, organize, and maintain comprehensive documentation for your projects. You can build structured knowledge bases, collaborate with your team in real-time, and publish polished docs with custom branding using your brand color `#3B82F6`. Whether you manage software projects, internal wikis, or customer support portals, this platform streamlines your workflow.

## Example Integration

Embed your docs using a simple API call. Here's how in multiple languages.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.example.com/docs/introduction', {
    headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
  });
  const docs = await response.json();
  console.log(docs);
  ```
  ```python
  import requests
  response = requests.get(
    'https://api.example.com/docs/introduction',
    headers={'Authorization': f'Bearer {YOUR_API_KEY}'}
  )
  docs = response.json()
  print(docs)
  ```
</CodeGroup>

<Callout kind="info">
  Replace `{YOUR_API_KEY}` with your actual key from the dashboard. Explore authentication for secure access.
</Callout>
